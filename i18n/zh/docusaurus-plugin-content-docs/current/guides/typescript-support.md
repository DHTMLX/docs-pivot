# TypeScript 支持 {#typescript-support}

DHTMLX Pivot 从 v2.0 起内置 TypeScript 类型定义，无需额外配置即可直接使用。

:::info
在 [Snippet Tool](https://snippet.dhtmlx.com/y2buoahe) 中在线体验 Pivot。
:::

## TypeScript 的优势 {#advantages-of-typescript}

类型检查和自动补全能够在早期发现错误，内置的类型定义可以清晰地告知您 DHTMLX Pivot API 所期望的数据类型。
