# API overview

## Pivot constructor

~~~jsx
new pivot.Pivot("#root", {
    // configuration parameters
});
~~~

**Parameters**:

- an HTML container (the ID of the HTML container)
- an object of the configuration parameters ([check here](#pivot-properties))
