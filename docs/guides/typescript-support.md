# TypeScript support

Starting from v2.0, DHTMLX Pivot supports TypeScript definitions. TypeScript support works out of the box.

:::info
Try out the functionality in our <a href="https://snippet.dhtmlx.com/y2buoahe" target="_blank">Snippet Tool</a>.
:::

## TypeScript advantages

TypeScript significantly increases development efficiency. Type checking and autocompletion make development more robust and help avoid mistakes. TypeScript also provides data type information for the DHTMLX Pivot API.
